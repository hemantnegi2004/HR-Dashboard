# HR-Dashboard


Goal: To analyze employee data and extract insights into workforce trends, promotions, retrenchments, job satisfaction, and overtime impact, aiding data-driven HR decision-making.

Code: HR Analytics Dashboard.pbix

Description: The dataset includes employee records containing details such as job role, years of service, promotion eligibility, retrenchment status, job satisfaction levels, and overtime hours. The project involves data loading, data cleaning, exploratory data analysis (EDA), and visualization through Power BI dashboards to uncover key workforce patterns and trends.

Skills: Data cleaning, data analysis, data visualization, HR analytics, dashboard design.

Technology: Power BI, DAX, Data Modeling.

Results: A comprehensive HR dashboard visualizing workforce distribution, promotion and retrenchment insights, job satisfaction levels, and overtime impact. The insights help HR teams identify areas for improvement, such as employee retention strategies and workload balance.
